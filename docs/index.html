Get started

One command. Five minutes.

Prerequisite: Ollama installed and running.

# Clone and run
git clone https://github.com/airblackbox/VaultMind.git
cd VaultMind
bash start.sh

# Or use Docker (no Ollama install required)
docker compose up

start.sh pulls the models (~4.5 GB, one-time), installs Python deps, starts the backend, and opens http://localhost:8000.
